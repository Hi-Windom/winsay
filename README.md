# Sofill-主题：内部代号 [ winsay ]（风颂）轻量化简洁风绛亽特色思源笔记主题

打包体积限定：小于3M

外观部分重写部分样式，依据绛亽有关标准和约定俗成。原则上不添加额外功能，功能强大的 Sofill= 主题 Coming Soon
